The Development Process

We started this project off by creating a restful server to host the St. Paul crime data. From there we continued on with creating the interactive map for our website using Leaflet. This piece was relatively simple to do as Leaflet had a pretty straightforward guide on how to place a leaflet map on your site. From there the project started to split and we started to tackle different parts of the project. For example, we got stuck on the search feature of the site and decided to tackle a different feature like using jQuery and vue to update our table. We got the map and table working, then implemented our filters for crime type, neighborhood, time, and date. Finally, we added Bootstrap to complete the website.
