The Development Process

We started this project off by creating a restful server to host the St.Paul crime data. From there we continued on with creating the interactive Map for our website using Leaflet. This piece was relatively simple to do as Leaflet had a pretty straightforward guide on how to place a leaflet map on your site From there the project started split and we started tackle diffrent parts of the project.
